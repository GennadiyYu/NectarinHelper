Это статическая страница. Бот принимает апдейты по вебхуку: /api/webhook (метод POST, только от Telegram).

Проверка здоровья

GET /api/health → 200 OK

Шаги настройки

  1. Добавь переменные окружения: BOT_TOKEN, MASTER_PASSWORD, TELEGRAM_WEBHOOK_SECRET, TELEGRAM_WEBHOOK_URL.
  2. Выполни локально: npm run set-webhook
  3. В Telegram отправь боту /start, затем /login Nectarinlove2022.
